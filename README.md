# Studyflows

A project made for a informatics school assignment.

The project explores:

- Web development
- Web design
- Backend development

The project uses the following technologies:

- React in the Vite framework, coded in typescript
- Supabase for the backend
- Vercel for hosting
- SQL for querying the database
